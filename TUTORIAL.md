# 《女人我最大》AI QA 編輯器使用手冊

這份手冊將幫助你快速上手 AI QA 產生器，輕鬆將文章轉換為高質量的問答內容並匯入 CMS。

## 1. 輸入文章 (Article Input)
1. 開啟 AI QA 產生器網頁。
2. 在左側的**文章輸入區**，直接貼上你需要轉換的原始文章內容。
3. 點擊「產生 QA」按鈕，系統會自動結合內部知識庫（RAG）進行分析並生成內容。

## 2. 檢視與編輯 (QA Output)
系統產出的內容會固定遵循**三個 H2 標題**的精簡結構：
1. **直接編輯**：在右側的 QA 輸出區，你可以點擊任何生成的問題 (Q) 或答案 (A) 直接進行文字修改，確保語氣符合資深編輯的標準。
2. **單筆複製**：每組問答旁邊都有專屬的「複製」按鈕，方便你單獨取用某一段內容。

## 3. 調整產出結果
如果你對當前的生成結果需要調整，可使用上方操作列的按鈕：
- **Regenerate All (重新產生)**：如果整體方向不對，點擊此按鈕（帶有微警告提示），系統會捨棄當前結果並重新生成一份新的 QA。
- **Sort Overview (排序總覽)**：點擊後可以快速檢視所有 QA 的大綱結構，並根據邏輯需求調整順序。

## 4. 匯出至 CMS (Export)
確認所有內容與排版都調整完畢後：
- 點擊畫面上最顯眼的 **Export JSON (匯出 JSON)** 按鈕。
- 系統會自動將你編輯好的 QA 內容打包成 CMS 支援的 JSON 格式檔案並下載。
- 將該 JSON 檔案直接匯入你們的 CMS 系統即可完成上稿。

---
**💡 溫馨提示**：
- 系統已內建最新版知識庫，**不需要**手動上傳任何 CSV 檔案。
- 請盡量利用系統自帶的「直接編輯」功能調整錯字或語氣，這比反覆重新產生更有效率。
